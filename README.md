## 專案啟動

動態資料與圖檔皆存放於 `app/publc/` 內，需先切換目錄至 `app/` 後以 `npm run start` 執行後端服務才可取得。

### 執行前端專案

```
npm install
npm run serve
```

### 專案打包

```
npm run build
```
