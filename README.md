## 技術規範
- [Tailwind.css](https://tailwindcss.com/)
- [Vue.js v2](https://v2.vuejs.org/)
- [CSS Paged Media](https://www.w3.org/TR/css-page-3/)



## 啟動專案

### 後端服務

動態資料與圖檔皆存放於 `app/publc/` 內，需先切換目錄至 `app/` 後以 `npm run start` 執行後端服務才可取得。

### 執行前端

```
npm install
npm run serve
```
